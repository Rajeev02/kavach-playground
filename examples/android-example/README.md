# Kavach SDK - Android Integration

This directory contains a skeleton integration of the Kavach Android SDK using Kotlin.

## Setup

1. Add the Kavach SDK to your `build.gradle` dependencies:
```gradle
implementation 'com.kavach.sdk:kavach-android:1.0.0'
```

2. Sync Gradle.

3. Initialize the SDK in your `MainActivity.kt` or custom `Application` class.
