# Kavach SDK - Python Integration

This directory contains a skeleton integration of the Kavach Python SDK, commonly used for backend services (FastAPI, Flask, Django) or scripts.

## Setup

1. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate
```

2. Install the SDK:
```bash
pip install -r requirements.txt
```

3. Run the example:
```bash
python main.py
```
